## Lune

Clean Next.js baseline for the rebuilt Lune project.

## Getting started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Start editing in `src/app/page.tsx`.
